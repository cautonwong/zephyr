# 智能电能质量监测仪（Power Quality Monitor）产品需求文档（PRD）

---

## 1. 产品概述与商业价值
在现代工业与智能电网中，非线性负载（如大功率变频器、LED 照明、数据中心服务器）向电网注入了大量的谐波，导致电能质量（Power Quality）恶化。电压暂降（Sag）、暂升（Swell）、闪变和高频谐波会造成精密半导体设备误动作、变压器过热烧毁，并面临电网公司高额的功率因数罚款。
本产品是基于万高 V32F20x 双核 SoC 研发的**“工业级边缘电能质量监测仪（PQM）”**。它通过高频同步采样与边缘实时 DSP，检测高达 50 次谐波，并在低于 1ms 的时间内识别出电压瞬态暂降，为有源电力滤波器（APF）和静止无功发生器（SVG）提供毫秒级控制补偿输入。

### 1.1 变现路径与商业模式
*   **谐波治理智能硬件销售（B2B）**：向电气设备制造厂、电能质量治理公司销售嵌入式 PQM 控制核心板（售价 400-600 元，毛利 $> 75\%$）。
*   **精密工厂用电质量诊断顾问 SaaS（B2B）**：为半导体、精密加工厂提供在线电能质量实时监测与退化诊断，按年收取“用电品质保障SaaS”订阅服务费，避免百万级停产损失。
*   **国网台区用电品质监控平台接入费（B2G）**：协助电力局监控分布式光伏接入后的网侧谐波超标问题，按接入终端点位收费。

## 2. 核心功能规格

| 序号 | 功能模块 | 功能描述 | 验收标准 |
|:---|:---|:---|:---|
| 1 | **超高频网侧采样** | M0 核心对三相电压、电流通道进行 12.8kHz 同步采样（每周期 256 点）。 | 采样同步相位误差 $\le 0.1^\circ$。 |
| 2 | **高阶谐波特征提取** | M33 运行 256 点基基 FFT，实时解算基波至 50 次谐波的幅值与相位，计算总谐波畸变率（THD）。 | THD 测量精度达到国标 Class A 级（误差 $\le 5\%$）。 |
| 3 | **亚毫秒级暂态捕捉** | 实时计算电压瞬时有效值（RMS），检测波形畸变与骤降（电压降至 90% 以下）。 | 暂降检测响应时间 $\le 1\text{ms}$ (1/2 周波)。 |
| 4 | **APF/SVG 补偿向量控制** | 实时生成瞬时无功功率与谐波补偿电流指令，控制片上 DAC 模拟输出驱动斩波功率级。 | 补偿电流解算延迟 $\le 200\text{us}$。 |

## 3. 技术实现方案与外设映射（基于 V32F20x）

### 3.1 三相通道与硬件 ADC 规划
*   **模拟前端（AFE）**：三相电压（Ua, Ub, Uc）与三相电流（Ia, Ib, Ic）通过外部互感器隔离后，送入 V32F20x 的片内双 ADC 控制器。
*   **ADC-DMA 链条**：由定时器（TIM0）作为触发源，驱动 ADC0 和 ADC1 进行同步多通道转换，结果直接通过 DMA 搬运至 `WAVEFORM_SHM` 中，M0 执行基础标定后发送 MBOX 中断给 M33。

### 3.2 50次谐波 FFT 算法实现
*   **基波锁相环（PLL）**：M0 运行软件锁相环，锁定工频 50Hz/60Hz，动态微调采样间隔，确保整周期采样，消除频谱泄露。
*   **M33 谐波分析**：
    *   M33 接收到一整周期采样数据后，调用 CMSIS-DSP 库的 `arm_cfft_f32` 进行 256 点复数傅里叶变换。
    *   计算 THD 因子：
        $$\text{THD} = \frac{\sqrt{\sum_{h=2}^{50}V_h^2}}{V_1}$$
    *   若 THD 超过 5%（国标限值），自动触发警告并记录至本地 FlashDB，同时通过 Modbus/Ethernet 上报。

## 4. 验收标准与测试矩阵
1.  **THD 精度校准测试**：注入包含 10% 3次谐波、5% 5次谐波的 220V 电压信号，验证 M33 计算输出的 THD 值为 $11.18\%$，误差在国标 Class A 限制范围内。
2.  **瞬时电压骤降响应测试**：人为触发波形突变使电压在一个周期内下降 30%，验证系统在 1ms 内向控制台输出 `[WARNING] Voltage SAG event detected!` 预警信息。
