# 商业化参考设计与套现矩阵 (Commercial Reference Designs PRD)

## 1. 愿景定义 (The Commercial Jewel)
将极致的嵌入式底层技术（多核、DSP、TrustZone、高精度ADC）转化为高溢价、高门槛的行业级参考设计（Reference Designs）。通过提供开箱即用的行业解决方案底座，实现从“卖芯片/卖硬件”到“卖系统/卖SaaS服务”的商业模式跨越。

## 2. 八大旗舰级参考设计矩阵 (The 8 Flagship Apps)

### 2.1 新能源与储能 (万亿级风口)
*   **`ref_bms_eis` (储能主动均衡 BMS 与 EIS 检测)**
    *   **核心技术：** 高频 DAC 激励注入 + 高精度 ADC 同步采样 + CMSIS-DSP 阻抗谱 (EIS) 分析。
    *   **商业价值：** 颠覆传统 BMS，提供电池真正的健康度 (SoH) 评估，赋能动力电池梯次利用。
*   **`ref_v2g_charger` (V2G 双向数字充电桩)**
    *   **核心技术：** 软件数字电源 + 并网 PLL + TrustZone/MCUBoot (ISO 15118 安全合规)。
    *   **商业价值：** 满足出海充电桩强制的高安全标准与电网双向交易需求。

### 2.2 具身智能与低空经济 (硬科技前沿)
*   **`ref_6dof_sensor` (机器人六维力矩传感器)**
    *   **核心技术：** 极低噪声微伏级 ADC 采样 + DSP 6x6 耦合矩阵解算 + 微秒级低延迟 IPC。
    *   **商业价值：** 突破人形机器人触觉瓶颈，用底层算法弥补国产应变片硬件的精度不足。
*   **`ref_drone_fcc` (多冗余工业飞控节点)**
    *   **核心技术：** ADRC 自抗扰控制 (抗乱风/断桨) + 双区 FOTA 容错 + 多核 AMP 隔离。
    *   **商业价值：** 面向 eVTOL 与工业无人机，提供符合航空级安全标准的底层固件中间件。

### 2.3 工业 AIoT 与数字资产 (SaaS 订阅模式)
*   **`ref_motor_pred_maint` (预测性维护边缘终端)**
    *   **核心技术：** 压电声学/振动采集 + TF-Lite Micro (边缘神经网络推理) + MFCC 提取。
    *   **商业价值：** 卖订阅服务，为化工厂/钢铁厂提供关键设备“永不停机”的预警监控。
*   **`ref_nilm_meter` (非侵入式边缘 AI 智能电表)**
    *   **核心技术：** 1024点高频 FFT 谐波分析 + CNN 负荷类型分类。
    *   **商业价值：** 一块电表识别全屋电器状态，为企业 ESG 碳盘查、商业节能改造提供数据底座。

### 2.4 泛医疗与智慧农业 (高壁垒特种行业)
*   **`ref_medical_wearable` (无创高精度生物电/光谱传感)**
    *   **核心技术：** 数字陷波器去工频 + Lock-in Amplifier (锁相放大器算法) 提取微弱光谱。
    *   **商业价值：** 医疗级精度的血氧/连续血糖监测算法底座，直接授权给消费电子大厂。
*   **`ref_nir_spectrometer` (近红外农产品快检仪)**
    *   **核心技术：** 光谱阵列采集 + Edge AI 偏最小二乘法 (PLS) 模型。
    *   **商业价值：** 替代昂贵实验室仪器，做成田间地头的便携式水果甜度/土壤成分检测仪。

## 3. 架构演进策略
所有的 `ref_*` 应用必须复用 `apps/vango_demo` 验证过的 IPC、Power、Security 基础服务。通过 CMake 的 Target 机制，实现业务模块的插拔。
